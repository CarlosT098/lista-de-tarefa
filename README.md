# Sobre o projeto

Desenvolvimento de uma lista de tarefas usando Java.

A aplicação consiste em armazenar uma lista as tarefas e as ordenando de acordo com sua prioridade.

# Tecnologias utilizadas
- Java
- JavaFX
- Maven
